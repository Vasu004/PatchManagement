Welcome to the Patch Management System

About the Project

Our Patch Management System is designed to handle and deploy software patches efficiently. The application provides a user-friendly interface to view and manage patches.

Key Features:

  • View all available patches
  • Deploy patches to the system
  • Track and manage patch status

We use Postman to manage and test the requests sent to the server. Postman helps us ensure that our APIs are functioning correctly and allows us to inspect the responses from the server.
